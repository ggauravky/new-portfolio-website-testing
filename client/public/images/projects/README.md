# Project Images

Place your project screenshots here!

## Naming Convention

- Use descriptive names: `chatbot-ai.jpg`, `data-dashboard.png`
- Use lowercase with hyphens
- Keep filenames short and clear

## Recommended Specs

- Size: 800x600px or 1200x900px
- Format: JPG, PNG, or WebP
- File size: < 500KB

## Usage in Database

```javascript
imageUrl: "/images/projects/your-image-name.jpg";
```
