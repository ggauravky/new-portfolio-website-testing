# Blog Cover Images

Place your blog cover images here!

## Naming Convention

- Use descriptive names: `ai-tutorial.jpg`, `web-dev-guide.png`
- Use lowercase with hyphens
- Keep filenames short and clear

## Recommended Specs

- Size: 1200x630px (social media optimized)
- Format: JPG, PNG, or WebP
- File size: < 500KB
- Aspect ratio: 1.91:1

## Usage in Database

```javascript
coverImage: "/images/blogs/your-cover-image.jpg";
```
