# Profile Photo Placeholder

**To add your profile photo:**

1. Take a professional photo (or use an existing one)
2. Recommended specifications:

   - Size: 400x400px or larger (square format)
   - Format: JPG or PNG
   - File size: Under 500KB (optimized)
   - Quality: Clear, well-lit, professional

3. Save it as: `profile.jpg`

4. Place it in this folder: `client/public/images/profile.jpg`

5. The Hero component will automatically display your photo

**Current fallback**: If no photo is found, a placeholder from placeholder.com is shown.

---

**Tips for a great profile photo:**

- Use good lighting (natural light works best)
- Clean, simple background
- Professional attire (business casual)
- Friendly, confident expression
- Make eye contact with camera
- Square crop (1:1 aspect ratio)
